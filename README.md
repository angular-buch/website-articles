# website-articles

This repo contains content for [https://angular-buch.com/blog](https://angular-buch.com/blog).  
Pull requests are welcome!

## Post Frontmatter

| Property       | Description                                                                                                      |
| -------------- | ---------------------------------------------------------------------------------------------------------------- |
| `hidden`       | Hide post in list                                                                                                |
| `sticky`       | Stick post to the top of the list                                                                                |
| `isUpdatePost` | Include post in separate "Updates" page. Applicable for regular "Angular X ist da!" blog posts and related stuff |
